# lex00's Homebrew Tap

A Homebrew tap with formulae for lex00 projects.

## Installation

```bash
brew tap lex00/tap
brew install chant
```

## Usage

After installation, you can use chant:

```bash
chant --version
chant --help
```

For more information about chant, visit: https://github.com/lex00/chant

## Formulae

- **chant** - Intent Driven Development tool for specification-driven development

### Chant

Chant is an Intent Driven Development tool that enables specification-driven development. Specs define work intentions, and the chant CLI executes them in isolated worktrees, ensuring reproducibility and auditability.

#### Installation

```bash
brew tap lex00/tap
brew install chant
```

#### Verification

```bash
chant --version
```
